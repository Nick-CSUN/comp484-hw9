Reflection:
The easiest part of the assignment was formatting and displaying the current date on the page. The hardest part was understanding how `Number()` works with different types of input, especially when values turn into `NaN`. I learned that the `Date` object requires adjusting the month since it starts at 0 and sometimes needs formatting like leading zeros. I also learned that the `Number` object can convert strings into numbers, but you have to check results using `Number.isNaN()` and `Number.isInteger()` to understand them. Finally, I learned how to display results in the browser by updating HTML elements using JavaScript and the DOM.
